YEG祭2026 会場案内　運営版 Ver.3.0
====================================

【公開ファイル】
GitHubリポジトリの一番上（ルート）へ、次の3ファイルをアップロードしてください。

  index.html
  .nojekyll
  vercel.json

ZIPファイル自体はアップロードせず、解凍した中身をアップロードしてください。

【一般公開】
通常のURLを開きます。
例：https://syoukoukaikaihatu.github.io/new-trail/

【管理者編集】
URLの最後に ?edit=1 を付けます。
例：https://syoukoukaikaihatu.github.io/new-trail/?edit=1

編集モードで変更できる項目：
・①～㉕の位置
・①～㉕の大きさ（番号ごとに個別設定）
・縁日ボタンの位置と大きさ
・TIMETABLEボタンの位置と大きさ

【編集手順】
1. ?edit=1 でページを開く
2. 移動したい番号またはボタンをドラッグする
3. 大きさを変えたい要素を選択する
4. 「選択中を小さく」「選択中を大きく」で調整する
5. 「変更版HTMLを保存」を押す
6. 保存された index.html をGitHubの既存ファイルへ上書きする

【重要】
・通常の公開画面では来場者は位置やサイズを変更できません。
・保存後のファイル名が index (1).html 等になった場合は、index.html に変更してからアップロードしてください。
・GitHub反映後は1～3分待ち、PCは Ctrl+F5、スマホはタブを閉じて開き直してください。

【主な仕様】
・添付の最終背景図面を使用
・25店舗対応
・縁日ボタンを押すと設置内容を表示
・TIMETABLEボタンを押すとステージ予定を表示
・出展者一覧は写真付きカードではないシンプル表示
・GitHub Pages / Vercel対応
・React等の外部フレームワーク不使用
